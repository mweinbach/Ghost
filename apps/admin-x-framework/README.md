# Admin X Framework

Ghost Shared Framework that is used by all the micro-frontends for common functionality like data fetching and routing.

## Pre-requisites

- Run `bun install` in Ghost monorepo root

## Develop

This is a monorepo package.

Follow the instructions for the top-level repo.
1. `git clone` this repo & `cd` into it as usual
2. Run `bun install` to install top-level dependencies.

## Test

- `bun run lint` - run just eslint
- `bun run test` - runs acceptance tests

In package.json you can find other related running options too.
